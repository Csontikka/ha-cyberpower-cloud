# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest  | Yes       |

## Reporting a Vulnerability

If you discover a security vulnerability in this integration, please report it responsibly:

1. **Do NOT open a public issue**
2. Use [GitHub Security Advisories](https://github.com/Csontikka/ha-cyberpower-cloud/security/advisories/new) to report privately
3. Include steps to reproduce the issue
4. Allow up to 72 hours for an initial response

## Scope

This integration communicates with the CyberPower PowerPanel Cloud API (`iotapi.cyberpower.com`). Security concerns related to:

- Credential handling and storage
- API communication
- Data exposure in logs or diagnostics

are within scope of this policy.
